Définition

la page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui est envoyée ( adresse en http://www.etc). C'est le navigateur qui interprête ce code.

Il existe deux sortes de balises :

  1. Les balises les plus courantes encadre du code qui se termine par une balise dite fermante. Ex : <balise> contenu HTML </balise>
  2. Les balises dites oprhelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple : <br/> <hr/> <img/> <imput/> etc :
    • balise i
    • balise u
    • balise em
    • balise mark
    • balise del
    • balise b
